# 按模型显示推理强度

## 目标

Codexbar 的推理强度菜单应与当前选中的 GPT-5.6 模型能力一致，并支持 Codex 新增的 `max` 与 `ultra` 档位。

## 设计

- 在 `CodexBarGlobalSettings` 中集中维护模型到推理强度列表的映射。
- `gpt-5.6-sol` 与 `gpt-5.6-terra` 支持 `low`、`medium`、`high`、`xhigh`、`max`、`ultra`。
- `gpt-5.6-luna` 支持 `low`、`medium`、`high`、`xhigh`、`max`。
- 未知模型继续显示基础四档，并保留当前配置中的自定义档位，避免破坏手工配置或未来模型。
- 菜单根据当前路由模型实时生成选项。
- 从支持更高档位的模型切换到能力较低的模型时，自动回落到目标模型支持的最高档位，并与模型设置一起持久化。

## 验证

- 单元测试覆盖三个 GPT-5.6 模型的能力表。
- 单元测试覆盖 `ultra` 切换到 Luna 后回落为 `max`。
- 单元测试覆盖受支持档位保持不变和不支持档位不能直接写入。
- 运行完整 macOS 测试，并构建、安装后检查菜单与配置写入。
